# 🐍 Python Mini Projects

A collection of beginner Python projects built to practice core programming concepts.

## 📁 Projects

### 🪨 Rock Paper Scissors
- Classic game against the computer
- Uses random module and conditionals
- Tracks win/loss/draw results

### 🔢 Number Guessing Game
- Computer picks a random number
- Player guesses with high/low hints
- Uses loops, conditionals and random module

### 🎲 Dice Rolling
- Simulates rolling a dice
- Uses random module
- Clean CLI output

## 🛠️ Tech Stack
- Python 3
- Random module
- CLI (Command Line Interface)

## ▶️ How to Run
1. Make sure Python 3 is installed
2. Navigate to any project folder
3. Run:
```
python filename.py
```
